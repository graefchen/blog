Introduction to this blog

11 February 2024

This Blog is mainly aimed for me to improve my understanding and knowledge about the topics I write about. Those can either be about video games, programming, computer science, linguistics and archaeology. And I hope that you as a reader of this blog will enjoy the articles that I will publish here.

How I made this blog:

This below is just something akin to an FAQ. These are informations about the process, the thoughts and the technology that helped me create this blog:

Articles: I have many different interests, but I try to vary my selection of themes. I can not guarantee a fair selection, as this blog is mainly aimed at myself and improving my skills in writing, may it be about some (boring) things about archaeology or programming.

Design: I aimed to make a blog more akin to the idea of the small web by Ben Hoyt. Furthermore I added dark and light modes, mainly because I personally prefer when a website changes colour according to the prefered mode.

Technology: I am using for this blog just plain old HTML and CSS and (currently) no JavaScript. Furthermore as I try to go for simplicity, there aren't any cookies. Additionally as there is no interactivity on this blog (at least for now), so the need to implement JavaScript or Cookies is not existing.
